# دليل خطة TCP50 — Earn2Trade

> **من التقييم إلى حساب مموّل** — دليل احترافي شامل لاجتياز حساب TCP50

---

## ما هو هذا المشروع؟

موقع تعليمي يشرح كيفية اجتياز تقييم **TCP50** من شركة **Earn2Trade** بطريقة منظمة واحترافية.

يغطي الموقع:
- مواصفات الحساب الأساسية (رأس المال · هدف الربح · حدود الخسارة)
- خطة الأرباح المقترحة على **10 أيام** مع رسم بياني تفاعلي
- قواعد إدارة المخاطر والحدود الشخصية
- **6 قواعد** لا يجوز مخالفتها أبداً
- توقيت يوم التداول بتوقيتي الشركة والجزائر
- آلية السحب وتقسيم الأرباح بعد الاجتياز
- الخطة النفسية للمتداول قبل وأثناء وبعد التداول
- أهم التحذيرات التي تؤدي لإلغاء الحساب

## التشغيل المحلي

### الطريقة 1 — فتح مباشر (الأسرع)
```
افتح ملف index.html مباشرة في المتصفح
```

### الطريقة 2 — سيرفر محلي مع npm
```bash
npm install
npm start          # يشغّل على http://localhost:3000
```

### الطريقة 3 — VS Code Live Server
ثبّت إضافة **Live Server** ثم اضغط **Go Live** من شريط الحالة.

### الطريقة 4 — Python (بدون تثبيت)
```bash
python -m http.server 3000
# افتح http://localhost:3000
```
